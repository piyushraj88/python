Big O values indicate how does the given algorithm scale for a given size of input. The common values and their implications are:

Big 0    Scalability

1	       Excellent

log n	  Good

n	      Average

nlogn	  Bad

n^2	    Very Bad

2^n	    Very Bad

n!	    Very Bad


Read further here: http://bigocheatsheet.com/
